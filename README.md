# JS Objects & Prototypes Lab

This repository contains exercises for the JavaScript Objects and Prototypes lab.

## Structure

- `exercise1.js` – Part 1: Function declarations & expressions  
- `exercise2.js` – Part 2: Arrow functions  
- `exercise3.js` – Part 3: Object literals (student)  
- `exercise4.js` – Part 3: Object literals (book)  
- `exercise5.js` – Part 4: Prototypes (vehicle & car)  
- `exercise6.js` – Part 4: Multi-level prototype chain (manager)  
- `exercise7.js` – Part 4: Method overriding (rectangle & circle)  
- `exercise8.js` – Bonus challenge: Library management system  

## How to run

Use Node.js to run each file in Command Prompt:

```bat
node exercise1.js
node exercise2.js
...
node exercise8.js
